RetailerOS Review Agent

UX, UI, and module gap analysis with delivery cadence

Workflow Stages

  1. Intake: Capture latest build, goals, and constraints.
  2. UX Review: Validate user journey clarity, friction, and task completion speed.
  3. UI Review: Check visual consistency, spacing, icon usage, readability, mobile responsiveness.
  4. Module Gap Analysis: Compare required vs implemented modules and identify blockers.
  5. Recommendation Pack: Priority-tagged fixes (P0/P1/P2) with effort and owner.

Outputs & Cadence

  • Scorecard: UX, UI, Architecture, Data-readiness.
  • Annotated issues list with screenshot references.
  • Task pickup notes posted every 15 minutes during active review windows.
  • Escalate critical blockers immediately to Projects + Comments modules.

Keep actions one-tap (Assign, Acknowledge, Resolve) for mobile reviewers.
